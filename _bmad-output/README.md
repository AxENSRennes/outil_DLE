# BMAD Output

This directory stores generated planning and discovery artifacts produced during BMAD workflows.

## Status

- keep this directory versioned for traceability
- do not treat it as the primary implementation entrypoint
- when the same topic exists in both `_bmad-output/` and `docs/`, `docs/` wins

## Intended Use

Use this directory to recover workflow history, generated drafts, and supporting artifacts that were not promoted into the canonical documentation set.

For day-to-day implementation work, start from:

- `docs/decisions/`
- `docs/implementation/`
- `docs/README.md`
